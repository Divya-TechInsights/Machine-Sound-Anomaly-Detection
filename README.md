## Overview  

**Machine Sound Anomaly Detection** is an AI-powered web application that identifies abnormal machine behavior based on sound patterns.  
By analyzing audio signals using **Deep Learning** and **Signal Processing**, this project helps in **predictive maintenance** ‚Äî detecting faults before they cause costly downtime.

##  How It Works  

1. **Upload a machine sound file** (in `.wav` format).  
2. The app extracts key **audio features** using `librosa`.  
3. The trained model analyzes the sound and classifies it as **Normal** or **Anomalous**.  
4. The result is displayed instantly along with relevant metrics.
